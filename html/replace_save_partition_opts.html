Inputs

  • Input Path: Required The UCE file to have its save partition replaced.
  • Part Path: Required The raw save partition file which will replace the UCE's existing one. Typically this is a 4Mb file called save.img. To work, it must have an internal ext4 filesystem. All UCEs generated with this and other tools have such a partition.
  • Backup Uce: Optional Select this option to create a backup of the UCE before any write operations happen. It will be created in the same directory as the UCE with the extension '.bak'.
