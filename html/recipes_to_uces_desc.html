Create UCEs from Recipes

Creates multiple UCE files from multiple 'recipe' directories. A recipe directory must contain sub-dirs called 'emu' and 'roms' as well as properly formatted 'exec.sh' and 'cartridge.xml' files. It can optionally have 'boxart' and 'save' sub-dirs, as well as a 'title.png' file. 'Recipe' dirs can be generated from roms using the appropriate operations selected from the main menu.

The tool can create custom save partitions. To do this include one of the following:

  • A file called save.zip containing a zipped save partition in the recipe root or 'save' subdir
  • A file called save.* - a raw ext4 filesystem, uncompressed, in the recipe root or 'save' subdir
  • Arbitrary files in a 'save/upper' subdir
  • Arbitrary files in the 'save' subdir. They will automatically be moved to 'save/upper' as required by the UCE format.
