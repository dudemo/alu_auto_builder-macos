Extract Save Partition

Extracts the save partition from an existing UCE file and saves it as a new file. This can then be inserted into another UCE file using 'Replace Save Partition'.

Inputs

  • Input Path: Required The UCE file from which to extract the save partition.
  • Output Path: Optional The filename/path for the extracted save partition. Defaults to 'save.img' in the same directory as the UCE.
