Inputs

  • Input Path: Required The gamelist.xml file from which to export assets.
  • Output Dir: Optional Location for the exported assets
  • Export Cox Assets Optional Export a folder of assets which can be directly copied to a USB stick for use wit CoinOpsX. Videos are not guaranteed to work as ATGames products are quite picky about video formats.
  • Export BitPixel Marquees Optional Export a folder of marquee files resized for use with the BitPixel. These are in addition to any full size marquees exported for CoinOpsX (if selected).
  • Do Summarise Gamelist Optional Choose whether to create files containing lists and a table showing what data/assets have been successfully scraped for each game. This is particularly useful when building recipes as it lets you determine which need further work before they are built into UCEs.
