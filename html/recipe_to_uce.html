Create UCE from Recipe

This operation requires administrator privileges on Windows

Creates a single UCE file from a single 'recipe' directory. A recipe directory must contain sub-dirs called 'emu' and 'roms' as well as properly formatted 'exec.sh' and 'cartridge.xml' files. It can optionally have 'boxart' and 'save' sub-dirs, as well as a 'title.png' file. 'Recipe' dirs can be generated from roms using the appropriate operations selected from the main menu.

The tool can create custom save partitions. To do this include one of the following in the 'save' subdir of a recipe directory:

  • A file called save.img - a raw ext4 filesystem.
  • A file called save.zip - as above but zipped up.
  • Arbitrary files in the correct folder structure. Usually all files go in a subdir called 'upper' with an empty dir called 'work' next to it.

Inputs

  • Input Dir: Required The properly constructed recipe dir.
  • Output Path: Optional A filename for the output UCE file. This can be generated if left blank.
